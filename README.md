# Aplikasi Data Sembako
Aplikasi untuk mencatat data sembako dan penjualan, dilengkapi dengan fitur import/export CSV, cetak laporan, dan sistem keamanan login dengan pertanyaan keamanan.

# Fitur Utama

-Tambah, edit, hapus data sembako/penjualan

-Fitur clear data massal

-Tabbed interface untuk multiple tabel

# Import & Export Data

-Upload data dari file CSV

-Download data ke format CSV

-Support untuk kedua tabel (sembako dan penjualan)

# Sistem Laporan

-Generate laporan menggunakan iReport

-Report Wizard untuk kemudahan penggunaan

-Cetak laporan langsung dari aplikasi

# Sistem Keamanan

-Login dengan username dan password

-Pertanyaan keamanan untuk reset password

-Entity class dengan persistence

# Berdasarkan pelaksanaan tugas ini, dapat disimpulkan bahwa:
1. Penggunaan JFrame dan JDialog Form memudahkan dalam merancang antarmuka pengguna yang interaktif dan terstruktur.
2. Entity Class berperan penting dalam mengelola persistensi data ke database secara efisien.
3. Integrasi iReport memungkinkan pembuatan laporan cetak yang rapi dan profesional.
4. Fitur import dan export CSV meningkatkan fleksibilitas dalam pertukaran data dengan sistem lain.
5. Tabbed Pane memudahkan pengguna dalam mengelola banyak data dalam satu jendela.
6. Fitur Clear dan Download memberikan kemudahan dalam manajemen data secara massal.
7. Implementasi login dengan pertanyaan keamanan berhasil meningkatkan lapisan keamanan aplikasi, terutama dalam hal pengubahan kata sandi/password.
